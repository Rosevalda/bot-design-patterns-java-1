***Design Patterns com Java*** 🚀

Este projeto foi desenvolvido como parte de um laboratório prático para explorar a implementação de Padrões de Projeto (Design Patterns) clássicos do GoF (Gang of Four) utilizando Java puro.

🛠️**Tecnologias Utilizadas**

•	Linguagem: Java 

•	IDE: Eclipse

•	Versionamento: Git & GitHub

🧩 **Padrões de Projeto Implementados**

O projeto foca em três categorias principais de padrões:

*1. Criacional: Singleton*

Garante que uma classe tenha apenas uma instância e fornece um ponto global de acesso a ela.

•	Implementações: SingletonLazy, SingletonEager e SingletonLazyHolder.

*2. Estrutural: Facade*

Provê uma interface simplificada para um subsistema complexo.

•	Objetivo: Abstrair integrações com Spring Data JPA e a API ViaCEP (via Feign).

Exemplo Prático: Simulação com a criação de dois subsistemas com dados como: CEP, Nome, Cidade e Estado.

*3. Comportamental: Strategy*

Permite definir uma família de algoritmos, encapsulá-los e torná-los intercambiáveis.

•	Exemplo Prático: Simulação de comportamentos de um robô (Normal, Defensivo e Agressivo).
